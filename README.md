# My Portfolio Website 
## Portfolio Website

This is my personal portfolio website, developed using **HTML** and **CSS** as one of my first front-end web development projects. The website is designed with a clean and simple layout to introduce myself and provide an easy way for visitors to get in touch.

### Features

* **Home Page:** Displays a brief introduction about me and my portfolio.
* **Contact Page:** Provides contact information for visitors to connect with me.
* Clean and responsive user interface.
* Built using only HTML and CSS without any frameworks or libraries.

### Technologies Used

* HTML5
* CSS3

### Purpose

The goal of this project was to strengthen my understanding of HTML and CSS fundamentals, including page structure, layouts, styling, navigation, and responsive design while creating a simple personal portfolio website.
